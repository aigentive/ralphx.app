# RalphX.app v0.2.1

RalphX.app 0.2.1 refreshes the frontend UI stack, updates native Rust security-related dependencies, and advances the build/test toolchain.

## Highlights
- Frontend styling and bundling refreshes Tailwind CSS to `4.2.4`, with `@tailwindcss/vite` and `@tailwindcss/postcss` updated to the same release. (`db513f5d`)
- Frontend data and rendering libraries refresh `@tanstack/react-query` to `5.100.1`, `lucide-react` to `1.11.0`, and `react-virtuoso` to `4.18.6` across query, icon, and large-list UI paths. (`db513f5d`)
- Native runtime dependencies refresh the Tauri backend’s security-oriented crate set, including `rustls-webpki` `0.103.13`, `time` `0.3.47`, `tar` `0.4.45`, `bytes` `1.11.1`, and `rand`. (`37d32fcd`)

## Fixes And Polish
- Frontend build and test tooling updates `vite` to `8.0.10`, `vitest` to `4.1.5`, and `typescript-eslint` to `8.59.0`. (`db513f5d`)
- React Query Devtools moves to `5.100.1` alongside the main React Query package. (`db513f5d`)

## Other
- This patch release is limited to dependency maintenance rather than new feature work. (`db513f5d`, `37d32fcd`)
