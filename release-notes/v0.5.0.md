# RalphX.app v0.5.0

RalphX.app 0.5.0 focuses on startup Git/GitHub auth recovery, macOS app polish, and safer release/visual validation.

## User-Facing Changes
- **Git/GitHub auth recovery** now runs at startup so auth problems are detected before agent workflows need `commit`, `push`, `fetch`, or PR creation; the warning remains visible until acted on or dismissed, with repair access in Repository settings and the Agents Commit & Publish panel. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)
- **Agents terminal startup** now uses a real terminal environment, avoiding common missing `TERM` and stripped Finder/Homebrew `PATH` cases without mutating user shell config. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)
- **macOS window chrome** is polished for the slim 48px custom header by re-centering the native traffic-light buttons on the visual midline. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)

## Fixes And Polish
- **Dialogs and chat context contracts** are hardened, including accessibility coverage for required dialog titles and descriptions. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)
- **README release presentation** is refreshed with a new framed Agents view hero image and a tap-backed Homebrew version badge linked to GitHub releases. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)

## Developer And Maintainer Changes
- **Release automation** now supports explicit exact-version and bump-type controls on the long-lived `0.x.y` track, including values such as `0.100.42`, while requiring manual approval for major bumps. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)
- **Release-note and proposal defaults** now use `gpt-5.5`, and release-note output is structured to prioritize user-facing changes separately from maintainer details. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)
- **Visual CI coverage** adds macOS frontend visual snapshot checks plus an aggregate frontend visual test check for visual-relevant changes, including startup, settings, and Commit & Publish git-auth repair surfaces. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)
- **Contributor guidance** now includes repository PR-description rules, with validation covering release scripts, workflow checks, dialog accessibility, git-auth flows, terminal behavior, and visual snapshots. [ca92c323](https://github.com/aigentive/ralphx.app/commit/ca92c32371d3ea4daee23b6d99eef58511da3e0a)
