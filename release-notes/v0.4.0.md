# RalphX.app v0.4.0

RalphX.app 0.4.0 improves installed macOS CLI resolution, enables the updater flow, separates dev/prod app runtimes, and refreshes the main app chrome.

## Highlights
- **Installed app runtime:** resolves git, gh, Claude, Codex, Node, shell, and process-management binaries through a shared production resolver so Finder/Homebrew launches are less dependent on an interactive terminal `PATH`. [5b781b85](https://github.com/aigentive/ralphx.app/commit/5b781b850edf71a358e8f7683f44407cf4a22d6a)
- **Updater flow:** enables Tauri update check/install permissions, registers restart after install, polls for same-day releases while RalphX remains open, and suppresses repeated notifications for the same version. [5b781b85](https://github.com/aigentive/ralphx.app/commit/5b781b850edf71a358e8f7683f44407cf4a22d6a)
- **Dev/prod coexistence:** lets an installed/Homebrew app run beside `./dev-fresh` by keeping production on backend `3847` / external MCP `3848` and defaulting dev to backend `3857` / external MCP `3858`. [5b781b85](https://github.com/aigentive/ralphx.app/commit/5b781b850edf71a358e8f7683f44407cf4a22d6a)
- **App chrome:** moves primary navigation into a left rail, slims the top header to 48px, re-centers macOS traffic lights, and updates affected sidebar/overlay offsets for the new layout. [5b781b85](https://github.com/aigentive/ralphx.app/commit/5b781b850edf71a358e8f7683f44407cf4a22d6a)

## Fixes And Polish
- **Dev MCP startup:** `./dev-fresh` now only cleans proven orphaned dev listeners, and the supervisor treats child `EADDRINUSE` as a terminal startup failure instead of retrying indefinitely against a stale ready endpoint. [5b781b85](https://github.com/aigentive/ralphx.app/commit/5b781b850edf71a358e8f7683f44407cf4a22d6a)
- **Dev backend routing:** centralizes frontend HTTP calls and MCP proxy backend URL routing so dev agents and UI talk to the dev backend instead of a running installed app. [5b781b85](https://github.com/aigentive/ralphx.app/commit/5b781b850edf71a358e8f7683f44407cf4a22d6a)
- **Visual baselines:** refreshes snapshots for the left-rail/header/logo layout and affected chat replay, agent artifacts, settings, loading, update-checker, and empty Kanban states. [5b781b85](https://github.com/aigentive/ralphx.app/commit/5b781b850edf71a358e8f7683f44407cf4a22d6a)
